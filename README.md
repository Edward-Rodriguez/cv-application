# CV/Resume Builder

Small application where users can input their information and generate a CV/résumé.

**Live Demo:** [`https://cv-application-d67.pages.dev/`](https://cv-application-d67.pages.dev/)

## Features

- Add multiple sections for education and work history
- Preview Template
- Edit/Clear entries

## Tech

- **Frontend:** React
- **Build Tool:** Vite
- **Styling:** CSS
- **Version Control:** Git & GitHub

## Deployment

- Cloudfare Pages

## Coming Soon / Future Additions

- Option to download template
- ATS friendly version option

## Install

git clone <https://github.com/Edward-Rodriguez/cv-application>  
cd cv-application  
npm install
